Pour utiliser notre code vous pouvez soit utiliser le fichier 'projet.ml' en mode interpréteur. Des tests y sont déjà insérés.
Pour tester notre code avec le flot d'entrée il faut ouvrir un terminal et exécuter le script 'flot_entree'. Ce script attendra
que vous inscriviez vos expressions. Une fois la suite de vos expressions écrites, faites entree puis Ctrl+D.
